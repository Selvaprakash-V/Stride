# Stride

Stride is a collaborative coding and interview practice platform used to run paired sessions with code execution, live video and scoring.

This repository contains a Node/Express backend and a React frontend built with Vite. See backend/ and frontend/ folders for details.

Getting started:

1. Install dependencies in `backend` and `frontend`.
2. Start backend (see `backend/package.json` scripts).
3. Start frontend with Vite.
